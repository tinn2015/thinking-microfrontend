# iframe

存在的问题：
* 子项目需要改造，需要提供一组不带导航的功能

* iframe嵌入的显示区大小不容易控制，存在一定局限性

* URL的记录完全无效，页面刷新不能够被记忆，刷新会返回首页

* iframe功能之间的跳转是无效的

* iframe的样式显示、兼容性等都具有局限性

* iframe 的跨域问题

采用iframe 的微前端框架 Mooa, Single-spa
